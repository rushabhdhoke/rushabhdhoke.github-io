Projects

Underwater Gesture Detection

Developing a an AI-based model for recognizing Scuba Divers’ Sign Language to facilitate underwater human-robot collaboration with (AUVs). Leveraging FastAI and ResNet-50, a pre-trained convolutional neural network, to create a deep learning model. Exploring opportunities for integrating the AI model with AUV control systems

Control of an Active Knee Orthosis

Replicated the Paper “Wilian M, et. al. Design and Control of an active knee orthosis driven by RSEA” in Simulation using Matlab Simulink, and Circuit design using EasyEDA. Developed new controller and obtained 87% accuracy.

Underwater Oyster Detection with AUV

Working with researchers from University of Maryland and Marine science Department towards a paper in Frontier Publication using BlueROV and AQUA2 AUV utilizing YOLOv5. Utilized Roboflow for annotations and classification, contributed in data analysis using obtained results for understanding results and collaborated with AI team on YOLO model training.

Fake News Detection Using ML

Developed a robust Machine Learning model to detect and classify fake news, leveraging key algorithms such as Linear Regression, Support Vector Machine, and Random Forest. Utilized Natural Language Processing techniques like tokenization, stop-word removal, and TF-IDF vectorization. Fine-tuned hyper-parameters of the SVM and Random Forest models using Grid Search and Cross-Validation, achieving an optimized balance between bias and variance.

KUKA LBR iiwa Robot Control

Developed a MATLAB-based control for the KUKA robot using Robotics System Toolbox. Derived DH Parameters for trajectory planning integrating obstacle avoidance and Aruco marker detection for precise navigation and manipulation.
